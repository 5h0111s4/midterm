以下の解説を全て確認された上で、それぞれの試験問題に解答してください。

解答するための事前準備

中間試験に解答するためには、次の準備を済ませておく必要があります。準備はこれまでの授業で全て済ませているはずですが、済ませていない場合は、解答する前に全て設定しておいてください。

  • Firefox Developer Edition のインストール
  • GitHub Desktop のインストール
  • Visual Studio Code といった、エディタ / IDE の用意
  • GitHub へのアカウント登録

特に GitHub のアカウント登録は、解答の提出のために必須となっています。登録されていない方は、こちらの登録ページからアカウント登録をしてください。

また登録されたアカウントを、清水にお知らせいただけないと、採点結果を成績に反映できなくなります。新たに GitHub のアカウント登録をされた方、もしくはこれまでにお知らせいただいていない方は、こちらのフォームにご記入ください。

解答方法

  • 試験問題は、01 から 10 までのフォルダに分かれています。
  • それぞれのフォルダ内の index.html を Firefox で開き、そこに書かれている要件を満たす関数を定義してください。
  • 関数の定義は、問題と同じフォルダにある js ファイルに記述してください。
  • 要件の充足率が、試験の点数となります。
  • 関数定義後、問題の記述されているページを再読み込みすることで、要件の充足度が測れます。

提出方法

  • 提出は GitHub で pull request を送ることによって行います。
  • pull request の送り方は、こちらの文書をご覧ください。
  • pull request が送られていない場合、受験されなかったものとして扱います。

問題

  • 恒等写像

    引数に指定された値をそのまま返す関数のことを恒等写像と言います。この問題は、恒等写像 id を定義します。

    この問題を解く

  • 偶数の判定

    自然数は、偶数と奇数の2種類に分類できます。この問題では、引数に指定された自然数がそのどちらに属するかを判定する関数を定義します。

    この問題を解く

  • FizzBuzz問題

    FizzBuzz というパーティーゲームがあります。このゲームのプレーヤーは順番に番号を言っていくのですが、特定の条件を満たす場合に、"Fizz"、"Buzz"、"FizzBuzz" と言わなくてはなりません。この問題では、引数に指定された番号の人が、言わなくてはならない言葉を返す関数を定義します。

    この問題を解く

  • 距離関数

    2つのものの間の距離を測る関数のことを距離関数と言います。この問題では、あるルールにしたがって2つのものの間の距離を返す関数を定義します。

    この問題を解く

  • うるう年

    1年の長さは365日ですが、時より366日となる年があります。そのような年のことをうるう年と呼びます。この問題では引数に指定された年が、うるう年か否かを判定する関数を定義します。

    この問題を解く

  • 進数変換

    私たちは数値を記述する時に、通常10進数で記述します。数字の記法には、10進数以外にも、2進数や16進数、60進数といったものがあります。これはそれぞれ2、16、60で桁がくりあがる記法です。この問題では10進数の数字を、7進数に変換するプログラムを記述します。
